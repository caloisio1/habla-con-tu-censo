Cómo funciona

Una respuesta auditable en cuatro pasos.

  1. Tu pregunta — la escribís en español, como hablarías con una persona.
  2. SQL validado — se traduce a una consulta SQL que pasa por un guardrail de seguridad antes de ejecutarse.
  3. Datos reales — se corre sobre los 3,3 millones de registros del Censo 2011; las celdas con menos de 5 casos se suprimen.
  4. Respuesta auditable — se redacta citando los números y siempre podés ver el SQL exacto que se ejecutó.
